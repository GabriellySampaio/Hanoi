1-JDK
    você precisa ter instalado o pacote JDK em seu computador.
    Você poderá instala-ló seguindo os passos desses tutorial, para Linux, 
https://www.devmedia.com.br/instalacao-e-configuracao-do-pacote-java-jdk/23749
    ou pesquisando no seu navegador por JDK.
2- Visual Studio Code
    Você precisa ter o Visual Studio Code em sua ultimá versão, para ambientes Linux basta instalar 
pela central de software.
    para outros sistemas operacionais, você poderá pesquisar por instalação Visual Studio Code no 
seu navegador.
3-Extensão VS
    Será necessário a instalação da extensão Java Extension Pack.
    Para isso basta você entrar em seu VS e procurar pelo ícone:
    ou, pressionar as teclas ctrl+shift+x.
    E pesquisar  pelo nome da Extensão, para garantir total funcionamento reinicie o VS.
4-Compilação e execução do projeto.
    Por fim basta abrir a pasta do projeto no VS e executar o arquivo Hanoi.java apertando a tecla F5.
